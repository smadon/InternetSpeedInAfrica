About Internet Speed In Africa

"Internet Speed In Africa" is a mapping project that will show various information on the connectivity in Africa. The goal is to provide a platform to see all the data related to Internet connectivity.
For now, you can see the average Internet Speed in all 54 african countries based on Akamai report. On the other layer, you can see information about terrestrial optical fiber, undersea cable fibre and Internet Exchange points.
More information will come over time (Internet Service Provider and different sources of Internet Speed). Stay tuned...
this web site was initially developed during a study project. the web site is hosted and supported by the Infomaniak

The source code for this website can be found on Github and is available under a General Public License
